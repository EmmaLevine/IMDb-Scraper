Pre-requisites:
A clone of the repository at https://github.com/EmmaLevine/IMDb-Scraper
download and install Node.JS and NPM from https://nodejs.org/en/download/

The output for ‘star trek’ can be viewed in the file scrape_results.txt
Once you run the tool, results will overwrite this file.

Run these commands in Terminal:
node -v
→ should show you the version
npm -v 
→ should show you the version
cd Documents/Code/IMDBWebScraper 
→ replace ‘Documents/Code/IMDBWebScraper’  with your local repository location
init -y
install axios
install cheerio
install async
install await
node index.js
→ Follow command line instructions
→ File will be saved to your local repository folder.
